# Loan Approval Prediction using Machine Learning

## Project Overview

This project uses Machine Learning to predict whether a loan application will be approved based on applicant information such as income, credit history, loan amount, and property area.

The objective is to assist financial institutions in making faster and more accurate lending decisions using predictive analytics.

---

## Dataset

* Source: Loan Prediction Dataset
* Records: 614 loan applications
* Features: 13 variables
* Target Variable: Loan_Status

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

---

## Machine Learning Model

Random Forest Classifier

### Model Performance

* Accuracy: 78.86%
* Loan Approval Rate: 68.73%

---

## Key Findings

* Credit History is the most important factor affecting loan approval.
* Applicant Income significantly influences approval decisions.
* Loan Amount impacts approval probability.
* Applicants with stronger financial profiles are more likely to receive approval.

## Business Recommendations

* Prioritize applicants with strong credit history.
